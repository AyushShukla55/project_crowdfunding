# Web3 Based Decentralized Crowdfunding Platform

## **Project Overview**
[span_1](start_span)[span_2](start_span)This project addresses the lack of transparency in traditional crowdfunding by using Ethereum Smart Contracts to automate fundraising and disbursement[span_1](end_span)[span_2](end_span).
